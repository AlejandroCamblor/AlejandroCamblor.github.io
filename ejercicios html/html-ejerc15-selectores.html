Tipos de selectores que entran en examen

Clase (class=””)

El identificador class en HTML sirve para asignar una o más clases a un elemento, lo cual permite aplicar estilos CSS o manipular elementos desde JavaScript de manera más organizada y eficiente.

Identificador (id=””)

El identificador class se coloca como un atributo dentro de la etiqueta HTML del elemento al que quieres aplicarle la clase.

Todos los elementos (*)

Párrafos (p)

La etiqueta p se utiliza para delimitar párrafos en html. Se coloca antes y despues de lo que queremos como párrafo.

Cursiva (em)

La etiqueta em sirve para poner en cursiva el texto que queremos. Se pone antes y despues de lo que queremos poner en cursiva

Títulos pricipales (h1)

La etiqueta h1 sirve para poner titulos del mayor tamaño posible en html. Se coloca antes y después de lo que queremos como título.

Segundos títulos (h2)

La etiqueta h2 sirve para establecer títulos de segundo nivel. Se coloca antes y despues del texto al que le queremos aplicar dicha etiqueta.

Elementos de una lista ordenada (ol li)

Estas etiquetas se usan para crear listas ordenadas (es decir, listas numeradas). Cada elemento dentro de una lista se representa con li, y todos esos elementos van dentro de la etiqueta ol.

Elementos de una lista no ordenada (ul li)

Estas etiquetas se usan para crear listas no ordenadas (es decir, listas no numeradas). Cada elemento dentro de una lista se representa con li, y todos esos elementos van dentro de la etiqueta ul.

Sobrevuleo (p:hover)

Sirve para aplicar un estilo especial cuando pasas el cursor por encima de un párrafo. p:hover se coloca dentro del bloque de estilos CSS, es decir, dentro de la etiqueta style que a su vez está dentro de la etiqueta head.
